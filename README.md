Integrated Medical Supply Chain Management System
A web-based platform built with Django to manage and automate the 
medical supply chain between a central company, agencies, and shops.

Problem
Medical agencies were managing stock and orders manually — through 
spreadsheets, calls, and paperwork. This caused delays, stockouts, 
and no visibility into where medicines were at any point.

What I Did
Built a role-based web application with three access levels — Admin, 
Medical Agency, and Medical Shop. Each role has its own dashboard 
and set of actions.

The admin can monitor the entire supply chain, approve orders, and 
view performance through Chart.js dashboards. Agencies can procure 
from the central company and manage shop requests. Shops can browse 
medicines and place orders, with full supplier transparency shown 
to customers.

Automated the order flow end to end — from procurement to delivery 
tracking — replacing all manual steps.

Tech Stack
Backend: Django, SQLite
Frontend: HTML, CSS, Chart.js

Results
- Automated 95% of supply tracking
- Reduced order processing time by 40%
- Achieved 99.5% data consistency across procurement operations
